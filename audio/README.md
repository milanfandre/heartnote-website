# Sample audio files

Drop the example song clips in **this folder** with these exact names and they
appear automatically in the "Hear a song" section on the home page:

| File name | Shows as |
|---|---|
| `sample-birthday.mp3` | Birthday · Country waltz — "Eighty Years Young" |
| `sample-wedding.mp3` | Wedding · Acoustic ballad — "The Day We Said Yes" |
| `sample-anniversary.mp3` | Anniversary · Soul — "Fifty Golden Years" |
| `sample-christmas.mp3` | Christmas · Warm carol — "Home for the Holidays" |
| `sample-faith.mp3` | Faith & Blessings · Hymn — "Grace for Our Family" |
| `sample-memory.mp3` | In Loving Memory · Tender piano — "Always With Us" |

The titles/occasions/styles are just labels — tell me if you want them changed
(they're set in the `SAMPLES` list in `index.html`).

## How to prepare each file

- **Format:** MP3 (plays on every phone/computer, including older devices).
- **Bitrate:** 128–160 kbps is plenty for web — keeps files small.
- **Length:** a **30–45 second excerpt** (the best chorus + a line or two), not
  the whole song — it loads fast and shows off the quality without giving the
  full product away.
- **File size:** aim for **under ~1 MB each** (a 40-second 128 kbps MP3 is ≈ 0.6 MB).
- **Volume:** normalize them so all six play at a similar loudness.

Until the files are here, each card shows a dimmed "coming soon" state — nothing
looks broken, it just won't play. Add a file and that card lights up instantly.
